Description

Documentation for the completion node. This node can be configured to use different language models as exposed by multiple services and offered for usage through Maya

You are required to obtain developer key from settings -> developers in the Maya application and set it in auth configuration to use it.

The cost of usage of this node depends on the service, model and number of tokens used in every request.

Config

authconfig
Create/select a authentication configuration to access language model.
Find your Maya developer key in Settings -> Developer
prompt string
Input prompt to be completed by the language model. Must contains tokens (approximate number of words) lower than the max token set.
modelstring
Select which language model to use for completion from the available models
temperaturenumber
Randomness to be introduced in the generated response. A number between 0 to 1 represents most determinitic and repetitive at 0 to least deterministic most creatiev at 1
max tokensnumber
Maximum number of tokens (approximately word of 4 characters) that can be present as as sum of input and output completion. The maximum for this value can vary between models and servies and are priced differently

Outputs

payload.promptstring
The input prompt sent to the model for completion
payload.completed string
The completion result returned by the selected language model
